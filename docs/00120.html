Running The Application

 

Running the Application : pmsm_foc_rolo_pic32_cm_mc - MCHV3 

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\pmsm_foc_rolo_pic32_cm_mc\firmware\mchv3_pic32_cm_mc_pim.X
  2. Clean and Build Project:

 

3. Program the project: 

 

4. Press the switch "Push Button" to start/stop the motor. 

5. Vary Potentiometer to change the speed of the motor. 

 

Fault Indication and Recovery : pmsm_foc_rolo_pic32_cm_mc - MCHV3 

If LED D19 is ON, it indicates a non-recoverable fault caused by either over-current, phase loss or due to erroneous rotor position estimation. The MCU needs to be reset in order to recover from this fault mode. 

 

 

Running the Application : pmsm_foc_rolo_pic32_cm_mc - MCLV2 

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\pmsm_foc_rolo_pic32_cm_mc\firmware\mclv2_pic32_cm_mc_pim.X
  2. Clean and Build Project:

 

3. Program the project: 

 

4. Press the switch S2 to start/stop the motor. 

5. Press the switch S3 to toggle motor direction.  

LED D2 Status 
Description 
OFF 
Motor spin direction is "positive" 
ON 
Motor spin direction is "negative" 

NOTE: Direction toggle command is accepted only when the motor is stationary. 

  1. Vary the Potentiometer to change the speed of the motor.

 

Fault Indication and Recovery : pmsm_foc_rolo_pic32_cm_mc - MCLV2 

If LED D17 is ON, it indicates a non-recoverable fault caused by either over-current, phase loss or due to erroneous rotor position estimation. The MCU needs to be reset in order to recover from this fault mode.
