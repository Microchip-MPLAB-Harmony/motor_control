Running The Application

Running the Application : pmsm_foc_smo_sam_e70 - MCLV2

  1. Open the project in MPLAB X IDE from : <install-directory>\motor_control\apps\pmsm_foc_smo_sam_e70\firmware\mclv2_sam_e70_pim.X
  2. Clean and Build Project

  1. Program the project

  1. Press the switch S2 to start/stop the motor.

5. Press the switch S3 to toggle the motor direction.

LED D2 Status 
Description 
OFF 
Motor spin direction is "positive" 
ON 
Motor spin direction is "negative" 

NOTE: Direction toggle command is accepted only when the motor is stationary. 

  1. Vary the Potentiometer to change the speed of the motor.

 

Project Name 
Output 
mclv2_sam_e70_pim.X 

  1. Press the Switch S2 to start the motor.
  2. Vary Potentiometer to change the speed of the motor.
  3. Observe waveforms on X2CScope scope view.
  4. Press the Switch S2 to stop the motor.
