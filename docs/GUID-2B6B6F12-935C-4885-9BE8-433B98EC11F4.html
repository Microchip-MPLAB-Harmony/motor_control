Microchip MPLAB® Harmony 3 Motor Control

MPLAB Harmony 3 is an extension of the MPLAB® ecosystem for creating embedded firmware solutions for Microchip 32-bit SAM and PIC32 microcontroller and microprocessor devices.This repository contains the MPLAB® Harmony 3 Motor Control module. This module contains motor control demonstrations implemented on Microchip's 32-bit SAM and PIC32 microcontrollers. Users can use these demonstrations as a reference for developing their own motor control applications on Microchip's SAM and PIC32 series of microcontrollers using MPLAB Harmony 3.
