1 Microchip MPLAB® Harmony 3 Motor Control

Microchip offers a range of software tools and demonstration projects designed to support motor control development processes. These software tools and demonstration projects are aimed at simplifying the implementation of motor control algorithms and accelerating time to market for engineers.

Software Tools

Microchip's Harmony QSpin tool offers engineers a user-friendly graphical interface to effortlessly configure motor control applications for their boards and motors. This powerful tool streamlines the configuration process and enables efficient setup of motor control applications. To learn more about Harmony QSpin and its features, please follow the link below:

  • Harmony Qspin

Demonstration Projects

Engineers can access and experience the motor control capabilities of Microchip devices through a range of demonstration projects. These projects serve as excellent starting points, providing hands-on examples of motor control implementation. Explore the links below to find demo projects designed for specific devices:

  • PIC32CM MC Motor Control applications
  • SAMC21 Motor Control applications
  • SAME54 Motor Control applications
  • PIC32MK Motor Control applications
  • SAME70 Motor Control applications
  • SAMRH71 Motor Control applications
  • SAMRH707 Motor Control applications

These demonstration projects, tailored for specific devices, provide valuable insights and practical guidance to engineers, enabling them to harness the motor control capabilities of Microchip devices effectively.
