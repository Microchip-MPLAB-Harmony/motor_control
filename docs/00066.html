Hardware Setup

Project mchv3_sam_e70_pim.X.

  • Hardware Used
    • MCHV3 Development Board (High Voltage)
      • Isolated Embedded Debugger Interface
      • ATSAME70 PIM
      • Leadshine EL5-M0400-1-24 Motor

 

  • Hardware Setup
  1. Mount the ATSAME70 Motor Control Plug In Module on U9 header.

  1. Place the "PFC - External Opamp Configuration" Matrix board at J4.

 

  1. Motor Connections:
    • Phase U - M1
    • Phase V - M2
    • Phase W - M3

 

 

  1. Encoder Connections:
    • A+ : HA
    • B+ : HB
    • +5V : 5V supply
    • GND : Ground

 

 

  1. Jumper Settings:
    • J11 - VAC ( Short Pin 3 - 4)
    • J12 - IA ( Short Pin 1 - 2)
    • J13 - IB ( Short Pin 1 - 2)
    • J14 - Fault_IP/IBUS ( Short Pin 1 - 2)

 

  1. Power the board with (110V/220V) AC mains. For additional safety, it is recommended to use a current limited power supply while testing this software demonstration on a non-default hardware and motor.

 

 

  • Installing Isolated Embedded Debugger
    • Default programmer or debugger daughter card shipped with the MCHV3 board cannot program or debug SAM series MCU and therefore, it needs to be replaced with an Isolated Embedded Debugger Interface for MCHV.

 

 

  • Complete Setup

  1. Project mclv2_sam_e70_pim.X.
    • Hardware Used
      • MCLV2 Development Board (Low Voltage)
      • ATSAME70 PIM
      • Long Hurst DMA0204024B101 Motor
    • Hardware Setup
  • Hardware Setup
    1. Mount the ATSAME70 Motor Control Plug In Module on U9 header.

  1. Place the "External Opamp Configuration" Matrix board at J14

  1. Connect motor and encoder terminals.

 

  1. Jumper Settings:
  • JP1 - Curr, JP2 - Curr, JP3 - Curr
  • In order to use RS232 port for X2CScope Communication JP4 - UART, JP5 - UART
  • In order to use USB port for X2CScope Communication JP4 - USB, JP5 - USB

 

  1. Power the board with a 24V DC supply using J2 or BP1-BP2. For additional safety, it is recommended to use a current limited power supply while testing this software demonstration on a non-default hardware and motor.
