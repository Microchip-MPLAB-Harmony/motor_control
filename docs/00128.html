pmsm_pfc_foc_pll_estimator_sam_e70

Power Factor Correction ( PFC ) shapes the input current drawn from the AC mains to be in synchronization with the input AC voltage. This project uses Boost-Converter topology to carry out the Power Factor Correction using Microchip's 32-bit SAME70 microcontroller in MCHV3.
