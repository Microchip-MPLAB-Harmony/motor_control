4.1 Setting up Custom Board

When working with Field-Oriented Control (FOC), using a custom board is often essential. A custom board allows you to tailor the hardware to your specific requirements, providing better insight into how FOC performs under your unique system constraints.

Follow the mentioned steps to configure peripherals for your custom board:

  • Click quick settings and select Custom from the "Development Board" drop-down menu.

  • Click on the PWM Interface block as shown below and configure it by following the steps mentioned here.

  • Click on the Analog Interface block as shown below and configure it by following the steps mentioned here.

  • Click on the Position Interface block as shown below and configure it by following the steps mentioned here.

  • Click on the Data Monitoring block as shown below and configure it by following the steps mentioned here.

  • Go to Pin Manager from Project Graph -> Plugins -> Pin Manager to set the pins.
